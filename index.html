  • Hand Controlled AI Virtual Mouse ▼

    This project replaces a traditional mouse with hand gestures using AI. Built with OpenCV, MediaPipe, and PyAutoGUI, it tracks hand movements via a webcam, allowing users to control the cursor and perform click actions without a physical mouse.

    How It Works:

    • The index finger moves the cursor.
    • Bringing the index finger and thumb together performs a click.
    • Uses MediaPipe for hand tracking and PyAutoGUI for mouse control.
    • A smoothening factor ensures fluid cursor movement.

    Tech Stack: Python, OpenCV, MediaPipe, PyAutoGUI

    Libraries: NumPy, Matplotlib, Pillow
